## 📊 文档内容概览

### BUG #1: Premature EOF 无限重试

**状态**: ⏳ 待修复 | **优先级**: 高

- **问题**: 后端文件删除后，FUSE 缓存仍保留原大小，导致 EOF 检查失败，重试无限循环
- **症状**: 每秒 10+ 条 `stream rw.Read nn=0, err=EOF` 日志
- **文件位置**: `vfs/read.go` 行 306-313
- **修复状态**: 等待 libfuse 3.18.1 升级验证后实施

### BUG #2: 父目录删除后仍然上传

**状态**: ✅ 已修复 | **优先级**: 高

- **问题**: 后端目录删除后，文件上传成功但重命名失败，导致无限重试
- **症状**: `error: failed to get src object: object not found` 每 5 分钟重复
- **文件位置**: `vfs/vfscache/item.go` 行 590-620
- **修复状态**: 2026-02-23 已在项目中实施
  - 添加了 parent directory 检查
  - 若目录不存在立即标记为干净，不重试
